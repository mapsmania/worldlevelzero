Welcome to the Four-Color Map Game!

The Four Color Theorem states that any map can be colored using at most four colors so that no two adjacent countries share the same color.

Rules of the game:

  • Select a color from the buttons.
  • Click a country to color it.
  • No two adjacent countries can have the same color.
  • Color all countries correctly to win!
